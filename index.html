Projetos

  • Rede de Sensores sem fios

    Orientação Eng. Lino Figueiredo (ISEP)

    Projeto para conclusão do mestrado no ISEP. Consiste numa rede de sensores sem fios com a tecnologia BLE. Criação do hardware, criação de PCB's para cada nó da rede, sensor+modulo BLE. Desenvolvimento do software para conectar 5 slaves a 1 master. O master tem conectividade com veículos (CAN-BUS) para obter informações da velocidade e RPM's. Toda esta informação é enviada para um servidor. Todo o sistema está orientado para IoT. Foi elaborado um paper sobre este projeto e apresentado na conferencia 2017 IEEE SOLI em Itália (Nota:19 valores)
